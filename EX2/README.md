# Experiment 2: Sorting, Searching and String Operations using 8086 Microprocessor

## Overview

This repository contains Experiment 2, which focuses on sorting, searching, and string manipulation techniques using 8086 Assembly Language. These programs are part of the Microprocessor Lab coursework and demonstrate fundamental data processing operations.

---

## Contents

### 8086 Microprocessor Experiments

* Sorting of array elements in ascending order
* Searching for an element using linear search
* Counting the number of elements in an array
* Comparison of two strings
* Finding the occurrence of a character in a string
* Substring search within a main string

---

## Concepts Used

* 8086 Assembly Language Programming
* Array traversal using SI register
* String instructions (CMPSB)
* Loop control using CX register
* Conditional jumps (JZ, JNZ, JBE)
* Memory addressing techniques
* String and array processing

---

## Tools Used

* vDos (DOS emulator for running 8086 programs)
* 8086 Trainer Kit (optional)

---

## Learning Outcomes

* Understanding sorting techniques in assembly language
* Implementing linear search algorithms
* Performing string comparison operations
* Identifying occurrences of characters in strings
* Implementing substring search logic
* Enhancing problem-solving using low-level programming
